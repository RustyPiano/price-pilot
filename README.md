# Price Pilot

比价格，选更划算的。

## 功能

- 添加商品，自动计算单价
- 支持多种单位（克、升、瓶、包等）
- 汇率换算（11种货币）
- 中英双语
- 浅色/深色主题
- 数据本地保存

## 快速开始

```bash
npm install
npm run dev
```

打开 http://localhost:3000

## 使用

1. 输入商品名称、价格、数量和单位
2. 点击"添加"
3. 系统自动计算单价并排序，最划算的排第一

## 技术

- Next.js 15
- React
- Tailwind CSS
- LocalStorage
